# RaidAttendance
A raid utility addon for World of Warcraft: Classic

```
/attend
/attendance    - Pops up a text box containing a CSV dump of the people online in guild and the people in raid.

/lootlog       - Toggles the logging of loot reception
/lootlog start - Enables the logging of loot reception
/lootlog stop  - Stops the logging of loot reception
/lootlog clear - Clears the current loot log
/lootlog reset - Stop logging loot reception and clears the current loot log

/lootdump      - Pops up a text box containing a CSV dump of loot reception and timestamps for rare or higher quality loot.
```
